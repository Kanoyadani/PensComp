Pensamento Computacional

1° - Decomposição

Decomposição seria usada da seguinte forma: o que precisa ser feito para criar o monstro? As crianças vão classificar os monstros pelo formato do rosto, procurar semelhanças nos monstros, fazer uma lista de características a serem identificadas e usar as características identificadas para criar um novo monstro.

2° - Abstração

A Abstração seria usada para identificar as diferenças entre os monstros e criar uma lista com todas as características diferentes entre eles.

3° - Reconhecimento de padrão

O padrão seria analisar se todos os monstros têm cabeças, olhos e outras características.

4° - Algoritimo

Por fim, o Algoritmo seria usado para criar uma série de instruções que podem ser seguidas pelos colegas de classe, como desenhar uma cabeça, olhos, nariz, orelhas e boca.

Sobre o Estudo

O pensamento computacional começará a ser ensinado nas escolas para crianças graças a Jeannette Wing. Mas como isso será ensinado? Cada grupo receberá um catálogo de monstros e cada monstro precisará ser classificado de acordo com as informações atribuídas a ele no catálogo. Primeiro, os alunos analisarão os monstros para ver o que todos eles têm em comum e, em seguida, abstrairão as diferenças para criar uma lista de instruções que podem ser usadas por todos para recriar um dos monstros do catálogo, sem saber de antemão qual está sendo feito!
